# 🌍 Gamified Environmental Education Platform

## 📖 Overview

The Gamified Environmental Education Platform is an interactive web-based application designed to promote environmental awareness through engaging quizzes and gamification elements. The platform makes sustainability education fun, interactive, and rewarding for students and learners.

## 🎯 Objectives

- Increase environmental awareness.
- Encourage sustainable practices through learning.
- Make environmental education engaging using gamification.
- Track user progress with scores, levels, XP, and rewards.

## ✨ Features

- Interactive Environmental Quiz
- Countdown Timer
- XP (Experience Points) System
- Level Progression
- Eco Coins Rewards
- Streak Tracking
- Eco Tree Growth Visualization
- Responsive User Interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Python
- Flask

## 🚀 How to Run

1. Clone the repository

```bash
git clone https://github.com/your-username/Gamified-Environmental-Education-Platform.git
```

2. Install Flask

```bash
pip install flask
```

3. Run the application

```bash
python app.py
```

4. Open browser

```text
http://127.0.0.1:5000
```

## 🎓 Educational Impact

This platform helps users learn environmental concepts through quizzes while motivating continuous participation using game mechanics such as points, levels, rewards, and achievements.
